### what I'm building

🕸️ **[arachne](https://github.com/walteraandrade/arachne)** — TUI git network graph viewer. Replaces GitHub's Network tab, right in your terminal. Rust.

🔭 **[mr-argus](https://github.com/walteraandrade/mr-argus)** — Linux process monitor built specifically for Claude Code instances. Keep tabs on your AI agents.

🎮 **[aetheria](https://github.com/walteraandrade/aetheria)** — Ear-training RPG. Vibe-coded into existence to make music theory actually fun.

🤖 **[review-summarizer-ai](https://github.com/walteraandrade/review-summarizer-ai)** — AI-powered opinion aggregator. Scrapes the web to summarize what people think about movies, albums, books, anything. Includes an interactive Influence Map.

📜 **[homer](https://github.com/walteraandrade/homer)** — Git chronicle TUI. Watches repos, groups commits into narrative units, generates prose. Go.

🚀 **apollo** — Local-first terminal app for commit review triage. Same spec, two AI agents, one winner. Go.
