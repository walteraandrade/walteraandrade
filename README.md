### ▶ Start here

🧰 **[integrated-task-manager](https://github.com/walteraandrade/integrated-task-manager)** — The triage layer Linear's free plan lacks. Ingests your error backlog, enriches each item from **Sentry/PostHog**, and an embedded **Claude** agent proposes an actionable verdict — nothing written back until you confirm. One **Deno 2** process (agent + store + scheduler + API/SSE) serving a React SPA, overlay on `node:sqlite`, zero external infra.
&nbsp;&nbsp;&nbsp;&nbsp;`TypeScript · Deno · React · MCP`

🔐 **[cerberus](https://github.com/walteraandrade/cerberus)** — A terminal password manager that takes crypto seriously: **XChaCha20-Poly1305**, **Argon2id**, secrets pinned in locked memory via memguard. Bubbletea TUI.
&nbsp;&nbsp;&nbsp;&nbsp;`Go`

🕸️ **[arachne](https://github.com/walteraandrade/arachne)** — A git network graph viewer, right in your terminal. Replaces GitHub's Network tab.
&nbsp;&nbsp;&nbsp;&nbsp;`Rust`
